# NETWithELK

## Tech

Tecnologia e Patterns utilizados:

- [.Net Core - 7.0] - Backend!.

## Installation

Executar docker-compose para instalar a stack do ELK.
Entrar na pasta  `NETWithELK.API`.

Executar o seguinte comando para executar o projeto.

```sh
dotnet watch run
```
